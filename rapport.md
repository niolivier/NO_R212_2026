# Rapport — [TD2]

## Points clés

### 1.Maitriser Flexbox pour la mise en page
J'ai appris à utiliser display: flex pour aligner des éléments. J'ai retenu que flex-direction: column permet d'empiler le titre au-dessus du menu, et que flex-wrap: wrap est indispensable pour que les colonnes s'adaptent toutes seules sur les petits écrans.

### 2. Harmonisation du design via le CSS
J'ai compris l'importance de la cohérence visuelle. En utilisant les mêmes codes couleurs (comme le gris clair #f0f0f0) pour les boutons et les fonds de témoignages, le site paraît beaucoup plus professionnel et fini.

### 3. Propreté du code et maintenance
J'ai appris à utiliser des variables (:root) pour gérer les couleurs et les espaces. Cela permet de modifier tout le site en changeant juste une ligne, ce qui rend le code plus facile à entretenir sans risque de tout casser.

J'ai utilisé Gemini comme assistant pour déboguer mon code et comprendre comment aligner mes éléments. L'objectif était de corriger mes erreurs de syntaxe et d'apprendre à utiliser les propriétés Flexbox.
